## Lovable Initial Prompt

Build a bilingual (DE/EN) Progressive Web App called **SAMVAD** — a structured conversation preparation tool for executive leaders and managers. Domain: samvad.purebeing.eu. Brand: PURE BEING.

**IMPORTANT — Paddle Compliance:** This is a SaaS productivity software product. Do NOT use the words "psychological", "therapy", "coaching", "diagnosis of people", or "pseudo-science" anywhere in the UI. Use "communication patterns", "structured preparation", "leadership reflection", and "conversation frameworks" instead.

**3. Generator — Step 1: Choose Conversation Framework**
- Title: `Welchen Gesprächsrahmen brauchst du?` / `Which conversation framework do you need?`
- Two options:
  - `Ich weiß es bereits` / `I already know` → show 11 framework tiles
  - `Hilf mir herausfinden` / `Help me find out` → go to Leadership Self-Reflection Tool
- 11 framework tiles: Motivieren, Ziele vereinbaren, Loben, Tadeln, Unangenehmes ansprechen, Kontrollieren, Konflikte versachlichen, Fördern, Fehler eingestehen, Verantwortungsdialog, Präsentieren

**4. Generator — Step 1b: Leadership Self-Reflection Tool**
Two-stage structured self-check. The leader reflects on their own actions — NOT an analysis of another person.

Stage 1 title: `Schritt 1: Habe ich meine Führungsverantwortung erfüllt?` / `Step 1: Have I fulfilled my leadership responsibility?`
Three Yes/No/Unsure questions:
- `Habe ich klar kommuniziert, was erwartet wird?` / `Have I clearly communicated what is expected?`
- `Habe ich die nötige Befugnis erteilt?` / `Have I granted the necessary authority?`
- `Habe ich Ressourcen und Mittel bereitgestellt?` / `Have I provided the necessary resources?`

Stage 2 title: `Schritt 2: Gesprächskontext` / `Step 2: Conversation Context`
One Yes/No/Unsure question:
- `Zeigt die Person Bereitschaft und Engagement?` / `Is the person showing willingness and engagement?`

Progress dots (4). Button: `Gesprächsrahmen ermitteln` / `Find my framework`

**5. Generator — Step 2: Context Input**
- Title: `Erzähl mir mehr.` / `Tell me more.`
- Subtitle: `Je mehr Kontext, desto präziser dein Leitfaden.` / `More context = more precise guide.`
- Three open text fields:
  - `Was ist das Thema?` / `What is the topic?`
  - `Warum kommt es zu diesem Gespräch?` / `Why is this conversation happening?`
  - `Was soll das Ziel sein?` / `What is the desired outcome?`
- Gold CTA button: `Leitfaden erstellen` / `Create Guide`

**6. Generator — Step 3: Your Guide (Result)**
- Gold badge at top: detected conversation framework type
- Sections:
  1. `Vorbereitung` / `Preparation` — mindset, key facts
  2. `Einstieg` / `Opening` — concrete opening sentence
  3. `Formulierungsbeispiele` / `Phrasing Examples` — 3–5 specific sentences
  4. `Worauf achten` / `What to Watch For` — communication pattern reminders, evaluation bias awareness, perception rules
  5. `Abschluss` / `Close & Commitment` — next steps, follow-up
- Bottom action row: `Speichern` / `Save` · `Teilen` / `Share` · `Als PDF` / `As PDF`

**7. Module: Welche Verantwortung fehlt? / Which Responsibility Is Missing?**
- 4-dimension leadership responsibility model (visual 2×2 grid)
- Standalone self-check for leaders
- Result: which dimension is unclear + recommended conversation framework

**8. Module: Kommunikationsmuster erkennen / Recognize Communication Patterns**
- Triangle diagram with three workplace interaction roles: Verfolger (Critic) / Opfer (Victim) / Retter (Enabler)
- `Häufige Muster` / `Common Patterns` pills: Ja-Aber, Schuldzuweisung, Gerichtssaal
- `Wie aussteigen?` / `How to exit?` checklist:
  - Erwachsenen-Kommunikationsmodus aktivieren / Activate adult communication mode
  - Sachebene ansprechen / Address the factual level
  - Verantwortung zurückgeben / Return ownership
- Self-reflection: `Welches Muster zeige ich gerade?` / `Which pattern am I showing?`

**9. Module: Vor dem Beurteilen / Before You Evaluate**
- Numbered list of 8 evaluation errors, each tappable with brief explanation
- Gold banner: `Nutze diese Liste als Pre-Flight vor jedem Beurteilungsgespräch.`

**10. Module: Kommunikations-Basics / Communication Basics**
- 2×2 grid:
  - `Kommunikationsstile` / `Communication Styles` (Parent/Adult/Child communication modes)
  - `Wahrnehmungsregeln` / `Rules of Perception`
  - `Quittungen` / `Acknowledgements`
  - `Fragetechnik` / `Questioning Techniques`
- Link card: `Insights für Leaders`

**11. Bibliothek / Library**
- Search bar + filter pills (by framework type)
- Saved guides list with Zen-style leadership thumbnails (diverse, international)
- `Insights für Leaders` section below with article cards

**12. Profil / Profile**
- Language toggle DE | EN

### Critical UI Rules

1. NEVER use the words "psychologisch", "Therapie", "Diagnose des Mitarbeiters", or "Coaching" in any UI label.
2. The Leadership Self-Reflection Tool is ALWAYS framed as the leader reflecting on themselves — never as analyzing another person.
3. The Drama Triangle module is labeled `Kommunikationsmuster erkennen` — never `Drama-Dreieck` in visible UI labels.
4. Transactional Analysis is labeled `Kommunikationsstile` — never `Transaktionsanalyse` in visible UI labels.
5. Follow the attached mockup images EXACTLY for layout, spacing, card sizes, and visual hierarchy.
6. The app must feel calm, spacious, and premium — never cluttered.
